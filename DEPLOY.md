# DEPLOY - Ready to Go!

Your GitHub username is already configured!

## Step 1: Push to GitHub

Open terminal in the `rindon-simple` folder and run these commands:

```bash
git init
git add .
git commit -m "Under construction page"
git remote add origin https://github.com/RinJohnson/rindon-simple.git
git branch -M main
git push -u origin main
```

**BUT FIRST** - Create the GitHub repo:
1. Go to https://github.com/new
2. Name it: `rindon-simple`
3. Make it Public
4. DON'T check any boxes
5. Click "Create repository"

Then run the commands above!

---

## Step 2: Deploy to Vercel

1. Go to https://vercel.com/new
2. Click "Import Project"
3. Find `rindon-simple` in your repos
4. Click "Import"
5. Click "Deploy"
